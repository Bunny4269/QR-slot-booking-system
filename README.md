# QR Slot Booking System

A Flutter-based QR Slot Booking and Verification System with Flask Backend.

## Features

- 9x9 Slot Booking Panel
- QR Code Generation
- QR Verification
- Admin & User Login
- Booking History
- Slot Verification
- Clear All Slots
- Real-time Backend Synchronization
- Cross Network Support

## Tech Stack

### Frontend
- Flutter
- Dart

### Backend
- Flask
- Python
